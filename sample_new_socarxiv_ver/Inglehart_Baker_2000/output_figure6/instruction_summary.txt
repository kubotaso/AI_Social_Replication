1. Identify countries with data in at least 2 WVS waves (waves 1, 2, or 3).
2. For each country, compute factor scores at each time point.
3. Plot arrows from initial position to latest position.
4. Open circle for initial, filled circle for latest.
5. Label each point with "Country YY" (YY = last 2 digits of year).

FACTOR SCORES: Must be computed on a common scale across all time points.
Option A: Compute factor analysis on pooled data from all waves, then score each country-wave combination.
Option B: Compute factor analysis on wave 3 (the reference), then project earlier wave data onto the same factors.
