Variable Mapping for Figure 7

PROCEDURE:
1. Use latest available survey for each country (wave 3 preferred).
2. Compute individual-level factor scores (or use country-level factor structure applied to individuals).
3. Create birth year cohorts from X003 (age) and S020 (survey year): birth_year = S020 - X003.
   Group into 7 decades: 1907-1916, 1917-1926, ..., 1967-1976.
4. Classify countries into 4 types (see figure_summary.txt).
5. For each society type and birth cohort, compute the mean factor score.
6. Weight each society equally (compute society mean first, then average across societies in the type).
7. Plot 4 lines.

SOCIETY TYPE CLASSIFICATION:
- Ex-Communist: N = 15,804 (use available ex-Communist countries)
- Advanced industrial: N = 21,947 (use available advanced industrial countries)
- Developing: N = 8,024
- Low-income: N = 5,280

Line styles:
- Ex-Communist: solid thick line
- Advanced industrial: dashed line
- Developing: dotted line
- Low-income: thin dashed/solid line with markers
