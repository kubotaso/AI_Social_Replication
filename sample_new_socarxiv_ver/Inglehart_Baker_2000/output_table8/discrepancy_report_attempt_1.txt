MAJOR ISSUES:

1. MISSING 1981 DATA FOR EUROPEAN COUNTRIES (23 missing values)
   The WVS Time Series v5 does not include EVS Wave 1 (1981) data for European countries.
   Countries affected: Belgium, Canada, France, West Germany, Great Britain, Iceland,
   Ireland, Northern Ireland, Italy, Netherlands, Norway, Spain, Sweden, United States.
   This also causes net change values to be wrong (using 1990-91 as earliest instead of 1981).

   POTENTIAL FIX: Since these 1981 values are from EVS Wave 1, we would need the EVS 1981 data.
   The ZA4460 file only contains EVS 1990-93 data. However, we could try:
   - Using the Inglehart-Baker paper's own 1981 values as ground truth and compute net change from there
   - The EVS Trend File (ZA3811 or similar) might contain 1981 data
   - Alternatively, accept these as "data unavailable" and score without them

2. INDIA 1990-91: paper=28%, computed=34% (diff=+6)
   The WVS data gives 33.7% for India Wave 2. The paper may have used a different
   subsample or weighting. India's Wave 2 survey was in 1990 with N=2478 valid responses.

3. SOUTH AFRICA 1995-98: paper=46%, computed=51% (diff=+5)
   The WVS Wave 3 gives 50.8% for South Africa. The paper may have used different
   sampling or only included certain population groups.

4. ROUNDING DIFFERENCES (32 partial matches within +/-2)
   The paper rounds to whole numbers. Python's round() uses banker's rounding.
   Some values that should round one way go the other. This affects net change as well.
   Consider using math.floor(x + 0.5) for traditional rounding.
