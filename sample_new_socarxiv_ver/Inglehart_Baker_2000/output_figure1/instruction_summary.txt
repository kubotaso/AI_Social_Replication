Variable Mapping for Figure 1: Global Cultural Map

PROCEDURE:
1. Compute the two factor scores (Traditional/Secular-Rational and Survival/Self-Expression) for each country using the 10-item factor analysis from Table 1.
2. Plot each country as a labeled point on a scatter plot.
3. X-axis: Survival/Self-Expression factor score
4. Y-axis: Traditional/Secular-Rational factor score
5. Draw hand-drawn style boundary lines to delineate cultural zones.
6. Add italic zone labels.

FACTOR SCORE COMPUTATION:
- Use nation-level data (country means of the 10 items).
- Run factor analysis with varimax rotation, 2 factors.
- Extract factor scores for each country.

CULTURAL ZONE BOUNDARIES:
- Use the original Figure1.jpg for reference to draw boundaries.
- Boundaries are smooth, freehand-style curves.
- Use matplotlib Path with CURVE4 for smooth Bezier curves.

AXIS RANGES: X from -2.0 to 2.0, Y from -2.2 to 1.8

COUNTRY LABELS: Each point should be labeled with the country name.
