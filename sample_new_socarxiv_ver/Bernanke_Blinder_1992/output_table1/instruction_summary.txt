================================================================================
MODEL SPECIFICATION
================================================================================

For each forecasted variable Y:

  Y_t = constant
        + sum_{j=1}^{6} beta_Y,j * Y_{t-j}
        + sum_{j=1}^{6} beta_CPI,j * log_cpi_{t-j}
        + sum_{j=1}^{6} beta_M1,j * log_m1_{t-j}
        + sum_{j=1}^{6} beta_M2,j * log_m2_{t-j}
        + sum_{j=1}^{6} beta_BILL,j * tbill_3m_{t-j}
        + sum_{j=1}^{6} beta_BOND,j * treasury_10y_{t-j}
        + sum_{j=1}^{6} beta_FUNDS,j * funds_rate_{t-j}
        + epsilon_t

Total regressors: 1 (constant) + 6*7 (lags) = 43

Estimation: OLS (ordinary least squares) for each equation separately.
This is NOT a VAR system — each equation is estimated independently.

================================================================================
F-TEST SPECIFICATION
================================================================================

For each column variable X in {M1, M2, BILL, BOND, FUNDS}:

  H0: beta_X,1 = beta_X,2 = ... = beta_X,6 = 0
  H1: At least one beta_X,j != 0

  Test: Standard F-test for joint significance of 6 restrictions.
  F(6, T-43) where T is the number of observations.

  Report: p-value of the F-statistic (marginal significance level).

Note: CPI lags are always included but CPI is NOT one of the tested columns.
The dependent variable's own lags are always included but not tested either.

================================================================================

8. Use statsmodels OLS or the f_test method to compute F-statistics and p-values.
