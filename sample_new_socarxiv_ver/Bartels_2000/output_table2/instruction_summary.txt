Instruction Summary for Table 2: Party Identification and Congressional Votes, 1952-1996

This file maps the paper's variables to the ANES Cumulative Data File (CDF) variables
for replicating Table 2. The specification is identical to Table 1 except the dependent
variable is congressional (House) vote rather than presidential vote.

================================================================================
DEPENDENT VARIABLE: Congressional (House) Vote
================================================================================

Dataset variable: VCF0707
Paper concept: Two-party House vote choice

Coding:
  VCF0707 = 1 -> vote = 0 (Democrat)
  VCF0707 = 2 -> vote = 1 (Republican)
  All other values (0, 3, 4, 5, 7, 8, 9, or missing) -> EXCLUDE from analysis

This restricts the sample to major-party House voters only.

================================================================================
INDEPENDENT VARIABLES: Party Identification Components
================================================================================

All independent variables are constructed from VCF0301 (Party Identification - 7-point scale).

Dataset variable: VCF0301
Paper concept: Party identification, used to construct three variables

VCF0301 coding:
  1 = Strong Democrat
  2 = Weak Democrat
  3 = Independent, leans Democrat
  4 = Independent (pure)
  5 = Independent, leans Republican
  6 = Weak Republican
  7 = Strong Republican
  0 or other = Missing -> EXCLUDE from analysis

--- Variable 1: "Strong" (Strong Partisan Identifier) ---

Measures the effect of strong party identification on vote choice.
Coded symmetrically so that strong identifiers of the respondent's party direction
are distinguished:

  VCF0301 = 7 (Strong Republican) -> Strong = +1
  VCF0301 = 1 (Strong Democrat)   -> Strong = -1
  VCF0301 = 2,3,4,5,6             -> Strong = 0

--- Variable 2: "Weak" (Weak Partisan Identifier) ---

Measures the effect of weak party identification on vote choice.

  VCF0301 = 6 (Weak Republican)   -> Weak = +1
  VCF0301 = 2 (Weak Democrat)     -> Weak = -1
  VCF0301 = 1,3,4,5,7             -> Weak = 0

--- Variable 3: "Leaners" (Independent Leaners) ---

Measures the effect of leaning partisan identification on vote choice.

  VCF0301 = 5 (Leans Republican)  -> Leaners = +1
  VCF0301 = 3 (Leans Democrat)    -> Leaners = -1
  VCF0301 = 1,2,4,6,7             -> Leaners = 0

--- Intercept: "Republican Bias" ---

The constant term in the probit model. A positive intercept indicates a pro-Republican
bias among pure Independents (VCF0301=4); a negative intercept indicates a pro-Democratic
bias. Pure Independents serve as the reference category since all three dummy variables
equal zero for them.

================================================================================
YEAR VARIABLE
================================================================================

Dataset variable: VCF0004
Paper concept: Election year

Include these 22 years (run a SEPARATE probit model for each year):
1952, 1956, 1958, 1960, 1962, 1964, 1966, 1968, 1970, 1972,
1974, 1976, 1978, 1980, 1982, 1984, 1986, 1988, 1990, 1992,
1994, 1996

NOTE: 1954 is NOT included. The ANES CDF variable VCF0707 (House vote) does not have
data for 1954, so that year is excluded. This results in 22 election-year models
(not 23).

================================================================================
SAMPLE RESTRICTIONS
================================================================================

For each year, the analysis sample consists of respondents who:
1. Have a valid VCF0707 value (1 or 2) - i.e., voted for a major-party House candidate
2. Have a valid VCF0301 value (1 through 7) - i.e., have a valid party identification
3. Belong to the election year being analyzed (VCF0004 = year)

================================================================================
STATISTICAL METHOD
================================================================================

Method: Probit regression (not logit)
Software: Use statsmodels Probit

For each of the 22 years, estimate:
  Pr(House vote = Republican) = Phi(b0 + b1*Strong + b2*Weak + b3*Leaners)

where Phi() is the standard normal CDF.

Report for each year:
- N (number of observations in the estimation sample)
- Coefficient and standard error for Strong
- Coefficient and standard error for Weak
- Coefficient and standard error for Leaners
- Intercept (constant) and its standard error
- Log-likelihood of the fitted model
- McFadden's Pseudo-R-squared

================================================================================
KEY DIFFERENCES FROM TABLE 1
================================================================================

1. Dependent variable: VCF0707 (House vote) instead of VCF0706 (Presidential vote)
2. Years covered: 22 years including midterm elections (1954 excluded)
   Table 1 covers only presidential election years (12 years: 1952-1996 by 4)
3. Everything else is identical: same independent variables, same probit method,
   same variable construction from VCF0301
